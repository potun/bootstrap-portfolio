Project information

  • Category: AR/VR/3D Design
  • Project date: 01 March, 2020
  • Project URL: www.modelviewer.dev

Model Viewer is an open source web component developed by Google and maintained through GitHub. Model Viewer aims at putting 3D content on the web easily with a few lines of HTML code. 3D models include low-poly and high-fidelity scans of Raul.
